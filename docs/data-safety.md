# Data Safety

This foundation does not include real user data, production API keys, or deployed cloud resources.

## Current Rules

- Mock data only
- Placeholder environment values only
- No AWS deployment
- No committed secrets
- No real user records

## Before Production

Define and document:

- Authentication and authorization flows
- Data retention periods
- Encryption requirements
- Audit logging
- Backup and recovery policies
- AI provider usage, consent, redaction, and retention rules
